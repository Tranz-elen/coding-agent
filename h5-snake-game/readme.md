## 🎮 障碍物系统特性

### 障碍物类型：
1. **墙 (Wall)**: 静态阻挡障碍物
2. **尖刺 (Spike)**: 伤害性障碍物，触碰减少生命值
3. **移动平台 (Moving Platform)**: 沿着路径移动的平台
4. **传送门 (Portal)**: 将蛇传送到随机位置
5. **加速区 (Speed Boost)**: 暂时增加蛇的移动速度

### 碰撞类型：
- **阻挡型**: 游戏结束或减少生命
- **伤害型**: 减少生命值
- **传送型**: 传送到新位置
- **加速型**: 暂时增加速度

### 智能特性：
- **路径跟随**: 障碍物可以沿着预定义路径移动
- **AI行为**: 支持空闲、巡逻、追逐、攻击等状态
- **动画系统**: 脉冲、旋转、颜色循环等动画效果
- **视觉效果**: 发光、纹理、渐变等渲染效果
